Flujo metodológico para generar líneas automáticas dentro de un polígono. El layout y conteo quedan administrables desde este panel.
Layout Operativo
0Capas / Mapas base por operación
Organización tipo QGIS por operación. Use este módulo para ortofotos JPG/PNG, KML/KMZ base, TIF de referencia y puntos/archivos operacionales. Los operadores solo ven las capas de su operación activa.
Para ortofotos use JPG/PNG + coordenadas WGS84. Para análisis topográfico use GeoTIFF EPSG:4326 en el módulo Topografía.
Sin capas.
